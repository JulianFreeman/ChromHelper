# ChromHelper

Chromium 核心浏览器辅助工具

## 说明

浏览器辅助工具（ChromHelper）目前有如下功能：

1. 浏览器插件检查
2. 浏览器书签检查
3. 浏览器设置检查
4. 导出书签到 KeePassXC 数据库
5. 追加插件
6. 清理缓存（仅支持 Windows）

支持的系统：Windows 10 以上，MacOS 10.13 以上

支持的浏览器：Chrome、Edge、Brave
